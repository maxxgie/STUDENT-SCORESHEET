Student Score Sheet

Introduction

This Java program is designed to manage and analyze student academic performance efficiently. It allows teachers to input scores for multiple subjects,
compute totals and averages, assign grades, and generate a formatted score sheet with class statistics.

Purpose
The purpose of this program is to:
1. Collect student scores in various subjects.
2. Calculate the total marks for each student.
3. Assign grades based on total marks.
4. Compute class-wide statistics such as subject totals, averages, and grade distribution.
5. Display the results in a clear and organized tabular format for easy evaluation.

---

Functionality
1. Input School Details:
   - The program prompts the user to enter:
     - School Name
     - Teacher Name
     - Grade
     - Academic Year

2. **Input Student Records:**
   - Supports input for 12 students (minimum requirement).
   - For each student, the following details are entered:
     - Name
     - Marks in English, Math, History, Geography, Science, and Programming

3. Compute Total Marks and Assign Grades:**
   - Total marks for each student are calculated by summing marks from all six subjects.
   - Grades are assigned as follows:
     - `A` ≥ 540
     - `B` ≥ 480
     - `C` ≥ 420
     - `D` ≥ 360
     - `F` < 360

4. Class Statistics:
   - Calculates totals and averages for each subject.
   - Counts the number of students achieving each grade.

5. Output:
   - Displays a formatted score sheet with:
     - Student names, individual marks, total, and grade
     - Totals and averages for each subject
     - Grade distribution across the class
